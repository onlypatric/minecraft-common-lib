# Checklist v0.3.0

## Obiettivo
Completare il blocco `commands + i18n advanced` come servizi riusabili per plugin multipli.

## Checklist
- [x] Aggiungere astrazione command model indipendente dal backend.
- [x] Aggiungere validazione input/permission model standard.
- [x] Migliorare `MessageService` con placeholder resolvers estendibili.
- [x] Supportare fallback locale multiplo configurabile.
- [x] Documentare guida migrazione da command raw Bukkit.

## Test obbligatori
- [x] Test parsing/validation command model.
- [x] Test fallback e pluralizzazione messaggi.
- [x] Test integrazione con scheduler/guard quando comandi eseguono task async.

## Exit criteria
- [x] Blocchi command+i18n pronti per adapter backend (CommandAPI/Cloud).
