# Checklist v0.1.0-alpha.2

## Obiettivo
Stabilizzare il core runtime già introdotto e preparare la base per freeze API `0.1.x`.

## Checklist
- [x] Congelare API pubbliche `dev.patric.commonlib.api` come baseline `0.1.x`.
- [x] Allineare Javadoc minima su tutte le classi core/runtime.
- [x] Garantire deprecazione esplicita per API legacy (`PluginLifecycle`, `Tasks`).
- [x] Verificare rollback fail-fast durante `onEnable()`.
- [x] Verificare cancellazione task su `onDisable()`.
- [x] Verificare fallback config/messages e gestione chiavi mancanti.
- [x] Mantenere porte future (`CommandPort`, `GuiPort`, `ScoreboardPort`, `ArenaResetPort`) come solo interfacce.
- [x] Aggiornare `CHANGELOG.md` e `COMPATIBILITY-MATRIX.md`.

## Test obbligatori
- [x] Unit test lifecycle, scheduler, service registry, config, messages, event router.
- [x] Plugin-like tests con MockBukkit.
- [x] Build completa verde (`clean test javadoc build`).

## Exit criteria
- [x] Nessun warning bloccante in build/javadoc.
- [x] Nessuna regressione su test suite esistente.
- [x] Versione pronta per passaggio ad `alpha.3`.
