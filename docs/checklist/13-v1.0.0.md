# Checklist v1.0.0

## Obiettivo
Rilasciare versione stabile `1.0.0` con API freeze e policy compatibilità definite.

## Checklist
- [ ] Freeze API pubblica e rimozione ambiguità nei package stabili.
- [ ] Stabilire policy compatibilità ufficiale (`N-2` minor Paper) documentata.
- [ ] Completare migration guide da `0.x` a `1.0`.
- [ ] Concludere audit performance/thread safety/licenze.
- [ ] Finalizzare docs complete: quickstart, cookbook, adapter matrix, troubleshooting.
- [ ] Definire release process ufficiale per minor/patch post-1.0.

## Test obbligatori
- [ ] Regression suite completa core + adapter.
- [ ] Smoke test su ambiente Paper reale con stack adapter principale.
- [ ] Verifica artifact e qualità javadocs pubbliche.

## Exit criteria
- [ ] Tag `v1.0.0` pronto e validato.
- [ ] Nessun blocker P0/P1 aperto.
- [ ] Libreria pronta per uso diffuso in plugin multipli.
