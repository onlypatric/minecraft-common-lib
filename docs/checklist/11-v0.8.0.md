# Checklist v0.8.0

## Obiettivo
Chiudere la prima wave di adapter reali (`CommandAPI`, `FastBoard`, `FancyHolograms`, `FancyNpcs`) mantenendo fallback no-op e capability detection robusta nel core.

## Checklist
- [x] Implementare adapter `CommandPort` (CommandAPI).
- [x] Implementare adapter `ScoreboardPort` (FastBoard).
- [x] Implementare adapter `HologramPort` (FancyHolograms) opzionale.
- [x] Implementare adapter `NpcPort` (FancyNpcs) opzionale.
- [x] Capability detection robusta (plugin presente/versione compatibile).
- [x] Fallback no-op trasparente quando adapter non disponibile.
- [x] Documentare setup server-side richiesto per ogni adapter.

## Test obbligatori
- [x] Integration smoke tests con e senza dependency adapter attiva.
- [x] Test graceful degradation in assenza dipendenze.
- [x] Test startup ordering (binding solo in enable path).

## Exit criteria
- [x] Adapter wave 1 pronta per adozione in plugin reali.
