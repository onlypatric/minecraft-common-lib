# Checklist v0.6.0

## Obiettivo
Introdurre match/state engine riusabile senza legare il core a un singolo gioco.

## Checklist
- [ ] Definire modello stato match (`lobby/countdown/running/ending/reset`).
- [ ] Aggiungere orchestrazione timer/event loop deterministica.
- [ ] Definire `EndReason` e cleanup contract.
- [ ] Definire policy rejoin e timeout di sessione.
- [ ] Mantenere match engine opzionale rispetto ai plugin non-minigame.

## Test obbligatori
- [ ] Test transizioni valide/invalid.
- [ ] Test rollback su errori in fase running/end.
- [ ] Test multi-match isolamento stato.

## Exit criteria
- [ ] Match engine pronto per arena/team/persistence.
