# Checklist v0.6.0

## Obiettivo
Introdurre match/state engine riusabile senza legare il core a un singolo gioco.

## Checklist
- [x] Definire modello stato match (`lobby/countdown/running/ending/reset`).
- [x] Aggiungere orchestrazione timer/event loop deterministica.
- [x] Definire `EndReason` e cleanup contract.
- [x] Definire policy rejoin e timeout di sessione.
- [x] Mantenere match engine opzionale rispetto ai plugin non-minigame.

## Test obbligatori
- [x] Test transizioni valide/invalid.
- [x] Test rollback su errori in fase running/end.
- [x] Test multi-match isolamento stato.

## Exit criteria
- [x] Match engine pronto per arena/team/persistence.
