# Checklist v0.9.0

## Obiettivo
Seconda ondata adapter e hardening su integrazioni a rischio maggiore.

## Checklist
- [x] Adapter claims (`HuskClaims`) con policy territory-safe.
- [x] Adapter schematic/reset (`WorldEdit` + `FAWE`) opzionali separati.
- [x] Adapter metrics (`bStats`) con relocation policy documentata.
- [x] Valutazione adapter packets (ProtocolLib/PacketEvents) con fallback sicuro.
- [x] Definire esplicitamente restrizioni licenza per adapter GPL/proprietari.
- [x] Completare matrice test esterna (claims, npc, boss, packets).

## Test obbligatori
- [x] Test handshake/version check per ogni adapter hard.
- [x] Benchmark reset arena e overhead event router.
- [x] Test stress multi-match con adapter attivi.

## Exit criteria
- [x] Libreria pronta per freeze API 1.0.
