# Release 2.1.0

Data: 2026-03-04

## Highlights
- Module System first-class in `api.module`.
- Dependency planning deterministico con cycle/missing detection.
- Lifecycle modulare `onLoad/onEnable/onDisable` con policy soft-disable.
- Diagnostica runtime moduli via `ModuleRegistry`.
- Compatibilita' preservata con `CommonComponent` legacy.
- Testing ground interno aggiunto: `examples/module-playground`.
- `SERVER-TEST` esteso con dump moduli (`CLIB_SMOKE_MODULES`) e report `modules.csv`.

## Nuove API
- `CommonModule`
- `ModuleState`
- `ModuleStatus`
- `ModuleRegistry`
- `ModulePlanResult`
- `CommonRuntime.Builder#module(...)`
- `CommonRuntime.Builder#modules(...)`
- `CommonRuntime.Builder#enableModuleDiagnostics(...)`
- `RuntimeBootstrap.build(JavaPlugin, Collection<? extends CommonModule>)`

## Note compatibilita'
- Nessuna integrazione con plugin consumer esterni in questa release.
- Nessuna dipendenza core aggiuntiva oltre baseline Paper `1.21.x` / Java `21`.
