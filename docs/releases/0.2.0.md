# Release Notes `0.2.0`

- Date: 2026-03-03
- Type: stable release
- Scope: plugin-generic ports + capability baseline

## Highlights
- Nuove porte plugin-generic ricche:
  - `NpcPort`
  - `HologramPort`
  - `ClaimsPort`
  - `SchematicPort`
- Aggiunto `PasteOptions` pubblico per operazioni schematiche.
- Capability model tipizzato:
  - `CapabilityKey`
  - `CapabilityStatus`
  - `CapabilityRegistry`
  - `StandardCapabilities`
- No-op ufficiali per tutte le nuove porte in `api.port.noop`.
- Runtime registra default no-op + capability `unavailable("No adapter installed")`.

## Test & quality
- Contract tests nuove porte.
- Behavior tests no-op.
- Test capability registry + runtime defaults.
- Enforcement no-external-deps rafforzato in `verifyCoreDependencyPolicy`.

## Limiti espliciti
- Nessun adapter reale implementato in `0.2.0`.
- Nessuna dipendenza esterna nel core.
- Modello embed-first invariato.
