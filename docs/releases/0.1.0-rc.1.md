# Release Notes `0.1.0-rc.1`

- Date: 2026-03-03
- Type: release candidate
- Scope: core runtime/plugin-generic only

## Included in RC
- Runtime kernel and deterministic lifecycle orchestration.
- Service registry, scheduler guardrails and tracked task cancellation.
- Config and message services for YAML + Adventure/MiniMessage.
- Event router with policy hooks.
- Bootstrap/error helper primitives for host plugin wiring.
- Contract tests for public API and legacy compatibility coverage.

## Explicit limits of this version
- No NMS layer in core.
- No external adapters in core (`CommandAPI`, `FastBoard`, `WorldEdit`, `FAWE`, `FancyNpcs`, `FancyHolograms`, etc.).
- Future ports remain interface-only (`CommandPort`, `GuiPort`, `ScoreboardPort`, `ArenaResetPort`).
- Async cancellation remains best-effort for tracked scheduler work; no preemptive cancellation of already-running async logic.

## Verification used for RC candidate
- `./gradlew --no-daemon clean test javadoc build`
- `./gradlew --no-daemon check`
- Focus tests:
  - `PublicApiFreezeContractTest`
  - `CommonRuntimeLifecycleTest`
  - `BukkitCommonSchedulerTest`
  - `LegacyApiCompatibilityTest`
  - `RcRuntimeBootstrapSmokeTest`
