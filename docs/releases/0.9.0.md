# Release Notes `0.9.0`

- Date: 2026-03-04
- Type: stable release
- Scope: adapter wave 2 + hardening

## Highlights
- Estensione binding model V2 in core:
  - `PortBindingService` ora copre `Claims`, `Schematic`, `BossBar`, `Metrics`, `Packets`
  - delegating wrappers aggiuntivi per fallback no-op trasparente
- Nuovi moduli adapter:
  - `adapter-huskclaims`
  - `adapter-worldedit`
  - `adapter-fawe`
  - `adapter-bossbar-paper`
  - `adapter-bstats`
  - `adapter-protocollib`
- Nuova API packet model (`api.packet/*`) + porta `PacketPort`
- Nuova API metrics (`MetricsPort`)
- Precedenza schematic backend enforced: `fawe` > `worldedit`
- Policy licenze hardening:
  - `verifyAdapterLicensePolicy` per moduli GPL/proprietari boundary-safe

## Runtime behavior
- Nessun adapter wave2 è hard-required dal core.
- Se backend assente/incompatibile: capability `unavailable(reason)` + fallback no-op.
- Se backend presente: bind atomico delegate + metadata capability `<backend>:<version>`.

## Limitazioni
- Nessun adapter PacketEvents in `0.9.0` (solo valutazione tecnica documentata).
- Nessuna pubblicazione Maven remota inclusa nel ciclo.
