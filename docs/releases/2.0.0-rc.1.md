# Release Notes `2.0.0-rc.1`

Data: `2026-03-04`

## Highlights
- GUI v2 API tipizzata (`GuiDefinition`, `SlotDefinition`, `GuiAction`, `GuiInteractionEvent`).
- Fluent DSL (`GuiDsl`) per GUI chest snelle.
- Slot policy engine (`BUTTON_ONLY`, `INPUT_DIALOG`, `TAKE/DEPOSIT`, `LOCKED`).
- Integrazione nativa con Dialog wrapper (`OpenDialogAction`) con state-binding:
  - `DialogResponseBinding`
  - `DialogOpenOptionsMapping.responseBindings`
- Nuove action first-class:
  - `ToggleStateAction`
  - `OpenSubMenuAction`
  - `BackMenuAction`
- Nuovo `GuiDefinitionRegistry` runtime per menu riusabili e navigazione stack/back.
- Nuovo adapter modulo `adapter-invui`.

## Compatibilità
- Paper `1.21.x`
- Java `21`
