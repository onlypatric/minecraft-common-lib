# Release Notes `2.0.0`

Data: `2026-03-04`

Release major focalizzata su GUI power-up.

## Include
- GUI DSL fluente e model slot/action tipizzato.
- Runtime interaction pipeline con policy e race safety.
- Dialog bridge da GUI con state-binding submit -> `GuiState`.
- Switch on/off first-class con dual-item rendering (`ToggleStateAction`).
- Submenu navigation stack + back (`OpenSubMenuAction`, `BackMenuAction`).
- Registry menu (`GuiDefinitionRegistry`) per composition modulare.
- InvUI adapter completo con backend reale (`InvUiGuiPort` su API InvUI `Gui`/`Window`).
- Dependency wiring InvUI centralizzato (`invuiVersion` in `gradle.properties`, repo XenonDevs in build root).

## Verification
- automatic gates:
  - `./gradlew --no-daemon clean test javadoc build`
  - `./gradlew --no-daemon check`
  - `./gradlew --no-daemon integrationTest -PrunIntegrationHarness=true`
  - `./gradlew --no-daemon adapterIntegrationTest -PrunAdapterIntegration=true`
- adapter-specific:
  - `./gradlew --no-daemon :adapter-invui:test`
- consumer demo:
  - `./gradlew --no-daemon -p examples/consumer-demo clean test -PcommonLibJar=../../build/libs/minecraft-common-lib-2.0.0.jar`
